# SmartEx Analytics Hub Server 🚀

Backend-сервис для аналитики, построенный на современном стеке технологий с упором на производительность и строгую типизацию.

## 🛠 Стек технологий

* **Язык:** Python 3.14.
* **Фреймворк:** FastAPI 0.135.1.
* **Менеджер пакетов:** uv.
* 
**База данных:** PostgreSQL 18 + SQLAlchemy 2.0 (Async).


* **Линтинг и форматирование:** Ruff.
* **Миграции:** Alembic.

## 🚀 Быстрый старт

### Требования

* Установленный `uv`.
* Docker и Docker Compose.

### Установка зависимостей

```bash
# Синхронизация окружения через uv
uv sync

```

### Запуск через Docker

```bash
# Сборка и запуск всех сервисов (API, DB, Redis)
docker-compose up --build

```

## 🛠 Разработка (Makefile)

Для удобства разработки используй команды из `Makefile`:

* `make lint` — запуск Ruff и MyPy для проверки кода.
* `make format` — автоматическое форматирование кода через Ruff.
* `make test` — запуск тестов через Pytest.

## 📝 Стандарт коммитов

В проекте используется **Conventional Commits**. Перед каждым коммитом автоматически запускаются хуки `pre-commit`:

1. Проверка `uv.lock`.
2. Линтинг и форматирование кода (Ruff).
3. Валидация сообщения коммита (Commitlint).

**Пример правильного коммита:** `feat(api): add data processing endpoint`.
